# Honor of Kings IMS

A simple Java information management system for Honor of Kings, supporting Players, Admins, Teams, and MatchRecords.

## Features
- Player and Admin account management
- Team creation and member management
- Match result recording
- Hero and equipment data support

## How to Run
Run `Main.java` under the `src/` directory using Eclipse.

## AI Usage
This project used a multi-agent workflow (Architect, Implementation, Review) to assist with class design and documentation.
